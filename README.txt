***SKYBOX README***
THIS APPLICATION CAN BE RUN BY ACCESSING THE INDEX.HTML FROM A WEB BROWSER
THIS CAN BE DONE DIRECTLY FROM A COMPUTER'S HARDDRIVE USING FIREFOX, 
CHROME WILL NOT WORK WITH THIS METHOD

TO RUN THIS APPLICATION IN CHROME, THE INDEX.HTML AND SUPPORTING FILES NEEDED TO
BE HOSTED ON A WEB SERVER AND ACCESSED THROUGH A HTTPS ADDRESS

AS AN OVERALL NOTE, THIS APPLICATION RUNS A LOT BETTER IN FIREFOX THAN CHROME
THE PREFERrED METHOD OF RUNNING THIS IS THEREFORE **FIREFOX**
